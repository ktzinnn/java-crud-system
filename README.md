User Management System (CRUD) - Java 💻
📌 Description

This project is a console-based CRUD application developed in Java.
It allows the creation, visualization, updating, and deletion of users, simulating a simple user management system.

The project was built to strengthen backend fundamentals and object-oriented programming concepts.

🎯 Project Objective

The main goal of this project is to practice and reinforce:

CRUD operations (Create, Read, Update, Delete)

Object-Oriented Programming (OOP)

List manipulation using ArrayList

Control flow and conditionals

User input handling with Scanner

Basic system organization

⚙️ Technologies Used

Java

Object-Oriented Programming

Java Collections (ArrayList)

Java Standard Library (Scanner)

🧩 Features

Create a new user with:

ID (auto-generated)

Name

Age

Email

Phone number

View user information

Update existing user data by ID

Delete a user by ID

In-memory data storage using List<User>

🏗️ System Structure

The application is structured with:

User class (entity)

App class (main logic and menu control)

Static ID counter for unique user identification

User search using ID comparison
